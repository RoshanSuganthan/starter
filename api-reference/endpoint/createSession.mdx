<Card title="POST /udm/v1/start/session" horizontal>
</Card>

Hosts can use this endpoint to initiate a new session, which is necessary before creating a questionnaire. The response will include a session ID, a URL, and an initial question.

Note: Hosts must include the session ID in subsequent questionnaire submissions to the provided URL.

Authorization - API key
