<Card title="POST /udm/v1/communication" horizontal>
</Card>

Hosts can use this endpoint to send an email that includes a cardholder letter. This feature is essential for ensuring that cardholders receive timely notifications and updates regarding their disputes.


Authorization - API key
---
 <ResponseField name="Key" type="string" required>
      x-api-key
    </ResponseField>

    <ResponseField name="Value" type="string" required>
     value
    </ResponseField>  


<RequestExample>

```bash request
curl --request POST \
  --url 'https://{{host}}/udm/v1/card-block' \
  --header 'Content-Type: application/json' \
  --data '{
    "recipient_email": "testuser@test.com",
    "subject": "Notification",
    "sender_name": "udm",
    "files": [
        {
            "file": "base 64 encoded data",
            "type": "pdf",
            "name": "sample.pdf"
        }
    ]
}'
```
</RequestExample>
