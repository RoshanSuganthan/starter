<Card title="POST /udm/v1/case/{{case_no}}/upload" horizontal>
</Card>

Hosts can use this endpoint to upload a document to a specific case within categorized groups.


Authorization - API key
---
 <ResponseField name="Key" type="string" required>
      x-api-key
    </ResponseField>

    <ResponseField name="Value" type="string" required>
     value
    </ResponseField>  


<RequestExample>

```bash cURL
curl --request POST \
  --url 'http://{{host}}/udm/v1/card-block' \
  --header 'Content-Type: application/json' \
  --data '{
    "files": [
        {
            "file": "base 64 encoded data",
            "type": "jpeg",
            "name": "name.jpeg",
            "category_id": "1"
        }
    ]
}'
```
</RequestExample>

Body
--- 

    <ResponseField name="Files" type="File Object">
        <Expandable title="properties">
        <ResponseField name="file" type="string" required>
        Base 64 encoded data string
        </ResponseField>

        <ResponseField name="type" type="string" required>
        Type of the file
        </ResponseField>

        <ResponseField name="name" type="string" required>
        Name of the file
        </ResponseField>

        <ResponseField name="category_id" type="string" required>
        Category for the file
        </ResponseField>
        </Expandable>
    </ResponseField>


Response
--- 

    <ResponseField name="status" type="enum" required>
      Status of the process. Allowed values - Success, Failed
    </ResponseField>

    <ResponseField name="message" type="string" required>
    Response message
    </ResponseField>
